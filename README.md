# DVWA Web Security Testing

This repository documents my hands-on practice with **Damn Vulnerable Web Application (DVWA)** to understand real-world **web application vulnerabilities** and how attackers exploit them.

The goal of this lab was to practice **OWASP Top 10 vulnerabilities** and understand how they can be discovered and exploited during **Web Application Penetration Testing (VAPT)**.
