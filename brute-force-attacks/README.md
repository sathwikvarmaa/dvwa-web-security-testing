# Brute Force Attacks

## Description
A brute force attack attempts to guess login credentials by repeatedly trying different username and password combinations.

## Objective
Demonstrate how weak authentication mechanisms can be exploited.

## Testing Method
- Identify login form parameters
- Intercept request using Burp Suite
- Send multiple credential attempts

## Impact
- Account compromise
- Unauthorized access

## Prevention
- Rate limiting
- Account lockout
- Multi-factor authentication
